# Python Data Analysis Projects

This repository contains selected Python exercises and projects completed as part of my MSc studies in Business and Economics (Data Science and Computational Economics) at the University of Basel.

The notebooks demonstrate practical experience in:

* Data manipulation and cleaning with Pandas
* Numerical and statistical computing with NumPy
* Data visualization
* Regression analysis
* Introductory machine learning methods
* Quantitative data analysis

### Exercise 05

### Exercise 09
